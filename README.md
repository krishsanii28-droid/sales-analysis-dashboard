# Sales Performance & Profitability Dashboard

 Project Overview
This project presents an interactive Tableau dashboard analyzing sales performance and profitability across different categories and regions.

## Key Features
- Dynamic parameter-based analysis (Sales vs Profit)
- Category-wise contribution analysis
- Sales vs Profit scatter plot for performance insights
- Heatmap visualization for regional trends
- Interactive filters for deeper exploration

## Key Insights
- Technology contributes the highest share of total revenue
- Profitability varies significantly across regions
- Some categories show high sales but lower profit margins

## Tools Used
- Tableau Public
- Excel (Data preprocessing)

## Files
- `sales-dashboard.twbx` → Tableau dashboard file

## How to View
Download the `.twbx` file and open in Tableau Public.
